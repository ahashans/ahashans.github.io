Hospital Management System

An Asp.net MVC web application for Hospital Management System. Four different role based user management has been implement with Microsoft's Identity Tool. A patient can purchase open ticket from a receptionist after successfull registration. A doctor can diagnose a patient with open ticket under his department. Patient can view his previous tickets and diagnosis. Doctor can view a patient's diagnosis report. Only an admin can create Receptionist and Doctor account and create new departments. He can also view list of all existing users. Click here to view the live project and click here to view the source code of the project in github

Morse Code Encoder & Decoder

An Arduino project for morse code decoder & encoder. In this project, text data is encoded and transmitted into audial and visual morse code format. And morse code is decoded and displayed in laptop display.

Push button is used in this project as morse code input device and laptop is used as text input and output device. An buzzer and LED is also used for audial and visual interpretion of morse code.
